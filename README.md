# Client

The frontend of my personal music portfolio. Built to showcase and share original music publicly.

## Overview

This is a Next.js application where I post and publish my music. Visitors can browse tracks, listen to songs, and explore my work. Media files are hosted via Cloudinary. The application is deployed on Vercel.

## Tech Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS
- Material UI
- Redux Toolkit
- Framer Motion
- Cloudinary (media hosting)
- Vercel (deployment)

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

| Command         | Description              |
| --------------- | ------------------------ |
| `npm run dev`   | Start development server |
| `npm run build` | Build for production     |
| `npm run start` | Start production server  |
| `npm run lint`  | Run ESLint               |

## Deployment

This application is deployed on Vercel. Every push to the `master` branch triggers an automatic deployment.

Live site: [gyanaprakashkhandual.vercel.app](https://gyanaprakashkhandual.vercel.app)
