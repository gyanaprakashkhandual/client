# Code of Conduct

## Overview

This is a private project maintained solely by Gyana Prakash Khandual. While the repository is publicly visible on GitHub, it is not open for external contributions.

This document outlines the expected behavior for anyone who views, references, or interacts with this repository.

## Expected Behavior

- Treat this repository and its owner with respect.
- Do not copy, reuse, or claim any part of this code as your own.
- Do not open issues or pull requests unless you have been explicitly invited to do so.
- Do not use the content of this repository to misrepresent or impersonate the owner.

## Unacceptable Behavior

- Plagiarizing or redistributing any part of the source code.
- Filing spam issues, pull requests, or discussions.
- Engaging in any form of harassment toward the repository owner.
- Attempting to exploit, scrape, or misuse any part of this application or its APIs.

## Reporting

If you believe someone has violated these terms or misused this codebase, please report it directly to the owner via GitHub.

## Scope

This Code of Conduct applies to all interactions with this repository, including issues, pull requests, discussions, and any external use of the code.
