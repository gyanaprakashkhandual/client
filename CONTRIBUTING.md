# Contributing

This is a private project and is not open to external contributions.

## Why

This repository is the personal music portfolio of Gyana Prakash Khandual. All development decisions, design choices, and code changes are made solely by the owner. There is no community roadmap, issue tracker for public use, or pull request review process.

## What This Means

- Pull requests from external contributors will not be reviewed or merged.
- Issues opened by the public will be closed without response unless specifically requested by the owner.
- Forking this repository for the purpose of contributing or redistributing is not permitted.

## If You Found a Bug

If you are visiting the live site at [gyanaprakashkhandual.vercel.app](https://gyanaprakashkhandual.vercel.app) and notice a bug or broken functionality, you are welcome to report it by opening a GitHub issue with a clear description. The owner may or may not act on it at their discretion.

## Development

All active development is handled privately by the owner. There is no public contribution workflow.
