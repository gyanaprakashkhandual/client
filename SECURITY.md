# Security Policy

## Scope

This document applies to the client-side frontend of the music portfolio at [gyanaprakashkhandual.vercel.app](https://gyanaprakashkhandual.vercel.app).

## Supported Versions

Only the latest deployed version of this application is maintained and supported. No older versions are actively patched.

| Version                    | Supported |
| -------------------------- | --------- |
| Latest (Vercel deployment) | Yes       |
| Older builds               | No        |

## Reporting a Vulnerability

If you discover a security vulnerability in this application, please do not open a public GitHub issue. Instead, contact the owner directly and privately via GitHub.

When reporting, please include:

- A clear description of the vulnerability
- Steps to reproduce the issue
- The potential impact or risk
- Any suggested fix, if you have one

## What to Expect

- The owner will acknowledge receipt of your report as soon as reasonably possible.
- Legitimate vulnerabilities will be investigated and addressed promptly.
- You will be notified once the issue has been resolved.

## Out of Scope

The following are not considered security vulnerabilities for this project:

- UI or cosmetic bugs
- Feature requests
- Issues with third-party services such as Cloudinary or Vercel

## Responsible Disclosure

Please act in good faith. Do not exploit any discovered vulnerability, access data you are not authorized to view, or disrupt the availability of the application.
